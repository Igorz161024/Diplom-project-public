# 📌 Changelog

У цьому файлі зберігається історія змін проєкту.

## [1.0.1] – 11 березня 2026
### 🔄 Changed
- Перехід на **PostgreSQL 18.3** (оновлення з 15‑ї версії).
- Виправлено шлях до volume для коректного збереження даних.
- Перевірено інтеграцію FastAPI‑бекенду з новою базою (ендпоінти `/add_entry`, `/get_account_balance`).
- Домовлено про використання **української латиницею** для назв рахунків, таблиць та описів.
- Додано `.env.dev` та `.env.prod` для безпечного зберігання конфігураційних змінних.

### ✅ Result
- Середовище стало більш **продакшн‑готовим**.
- Бекенд успішно працює з PostgreSQL 18.3.
- Конфігурація тепер гнучка й безпечна завдяки `.env` файлам.
- Дані відображаються стабільно у PowerShell без проблем із кодуванням.
